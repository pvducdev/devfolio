# My Projects

Here are some of the projects I've worked on:

## Project 1: Tab System

A modern tab system built with:

- **TanStack Start** for routing
- **Zustand** for state management
- **MDX** for content
- **Vite** for blazing fast builds

### Key Features

- ✅ Lazy loading of content
- ✅ Persistent tabs across refreshes
- ✅ Modern MDX v3 with full TypeScript support
- ✅ Build-time compilation for optimal performance

## Getting Started

To use the tab system in your project:

```bash

```

Then configure Vite and start creating MDX files!
